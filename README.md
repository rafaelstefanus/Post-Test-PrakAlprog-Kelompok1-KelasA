# Post-Test-PrakAlprog-Kelompok1-KelasA
Post Test Analisis Gender Inequality Index (GII). Praktikum Algoritma &amp; Pemrograman Kelas A Kelompok 1.

## Deskripsi Singkat

Repository ini berisi analisis data mengenai **Gender Inequality Index (GII)** yang bertujuan untuk memahami hubungan antara pembangunan manusia, pendidikan perempuan, kesehatan ibu, dan ketimpangan gender.

Disini dilakukan proses pengolahan dan analisis data mulai dari tahap persiapan dataset, eksplorasi data, visualisasi, hingga penyajian hasil analisis dalam bentuk grafik dan infografis agar informasi lebih mudah dipahami.

### Isi Repository

Repository ini berisi beberapa komponen utama:

- **Dataset**
- **Source Code**
- **Hasil Visualisasi Grafik**
- **Infografis**
