# Post-Test-PrakAlprog-Kelompok1-KelasA
Post Test Analisis Gender Inequality Index (GII). Praktikum Algoritma &amp; Pemrograman Kelas A Kelompok 1.

## Deskripsi Proyek

Repository ini berisi analisis data mengenai **Gender Inequality Index (GII)** yang bertujuan untuk memahami tingkat ketimpangan gender berdasarkan dataset yang ada.

Dalam proyek ini dilakukan proses pengolahan dan analisis data mulai dari tahap persiapan dataset, eksplorasi data, visualisasi, hingga penyajian hasil analisis dalam bentuk grafik dan infografis agar informasi lebih mudah dipahami.

### Isi Repository

Repository ini berisi beberapa komponen utama:

- 📁 **Dataset**
  - Berisi data mentah yang digunakan dalam proses analisis Gender Inequality Index (GII).

- 📁 **Source Code**
  - Berisi kode program yang digunakan untuk melakukan proses:
    - Import dan pengolahan data
    - Pembersihan data (data preprocessing)
    - Analisis data
    - Pembuatan visualisasi grafik

- 📁 **Hasil Visualisasi**
  - Berisi grafik hasil analisis data yang digunakan untuk melihat pola, perbandingan, dan informasi terkait GII.

- 📁 **Infografis**
  - Berisi rangkuman hasil analisis dalam bentuk visual yang lebih sederhana dan informatif agar mudah dipahami oleh audiens umum.
